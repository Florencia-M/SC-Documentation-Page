Introducción

La página a la que se hace alución en éste resumen es Landing Page por Florencia-M

Creada como parte de uno de los 5 proyectos necesarios para la certificación de Responsive Web Design.

A continuación se dan detalles sobre la estructura de la web, el producto elegido, los objetos que componen la imágen de marca y cómo se desarrolló.

Estructura

Está compuesta por un sólo index html dividido en 3 secciones, enlazadas a través de IDs a la barra de navegación.

Cumple con los 15 user stories requeridos, detallados en la siguiente lista:


  1. Un elemento header con su correspondiente ID="header".
  2. Imágen con el logo de la marca dentro del elemento del punto anterior, con una ID="header-img".
  3. Un elemento NAV con un ID="nav-bar" dentro de #header.
  4. Tres elemento clickeables incluídos en el elemento NAV cada uno con la clase "nav-link".
  5. Ser llevado a la sección de la web que corresponda a cada uno de los links anteriores al clickear sobre ellos.
  6. Un elemento video con el ID="video".
  7. La web debe incluír un elemento form con su ID="form"
  8. Dentro del elemento form incluír un campo input con el ID="email" donde se pueda escribir el mismo.
  9. Placeholder donde el usuario pueda leer instrucciónes sobre lo que le es requerido hacer en input
  10. El elemento #email debe usar una validación HTML5 para confirmar que el texto ingresado en él sea efectivamente una dirección de email.
  11. El elemento form debe contener un submit con su correspondiente ID="submit"
  12. Cuando se clickea #submit el email es enviado a una página estática con una URL ya proporcionada.
  13. La barra de navegación debe estar siempre en la cima a la vista del usuario.
  14. El Landing Page debe de tener un media query.
  15. Se debe utilizar CSS flexbox al menos una vez.

Lenguajes utilizados:


Para darle estructura a la web se utilizó HTML, lenguaje de marcado. Y CSS plano.

Producto

La Landing Page en cuestión se refiere a una cafetería de origen Italiano que llega a Uruguay, Costa de Oro; donde también se puede conseguir libros.

Se presenta una breve descripción de la misma y una muestra de 3 de sus productos (Café, cereales con fruta y un postre) a través de una pequeña galería, más un video mostrando granos de café.

Imágen de marca

Sc Coffee Shop es una marca ficticia, creada para el proyecto. Todas las direcciónes, teléfonos, etc.. son igual de ficticias.

Los logos también fueron creados para el proyecto, con la herramienta de diseño Canva. Las fuentes de letra elegidas son Amsterdam Four y UNICA ONE. En colores blanco y negro.

Las fuentes importadas desde GoogleFonts para la web son UNICA ONE y Dancing Script.

Con la idea de dar un aspecto 'cálido' y relacionado con el producto, se utiliza un background en tonos café.

Desarrollo

El mismo fue de corta duración en cuanto código, alrededor de 3,5 horas. El diseño de los Logos y contenido multimedia 2 horas.

Se optó por un diseño responsive, apto tanto para PC como para dispositivos móviles y evitar todo tipo de librerías externas exceptuando 2 fuentes de Google.

El contenido multimedia está alojado en la nube de Dropbox para el correcto display de las mismas sin necesidad de un servidor.
